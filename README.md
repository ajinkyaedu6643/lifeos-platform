# LifeOS

LifeOS is a personal life management platform designed to manage health, finance, knowledge, projects, freelance work, documents, and family activities from a single application.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Maven

## Current Features

- User Management
- Role Management
- Basic Authentication
- DTO Validation

## Roadmap

- Workspace Management
- Person Management
- Document Vault
- Investment Tracking
- Knowledge Management
- Passbook Vault Integration
- AI Assistant
